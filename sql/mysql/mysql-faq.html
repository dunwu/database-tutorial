      # 1. СИ║С╗ђС╣ѕУАеТЋ░ТЇ«тѕаТјЅСИђтЇі№╝їУАеТќЄС╗ХтцДт░ЈСИЇтЈў

      сђљжЌ«жбўсђЉТЋ░ТЇ«т║ЊтЇаућеуЕ║жЌ┤тцфтцД№╝їТѕЉТііСИђСИфТюђтцДуџёУАетѕаТјЅС║єСИђтЇіуџёТЋ░ТЇ«№╝їТђјС╣ѕУАеТќЄС╗ХуџётцДт░ЈУ┐ўТў»Т▓АтЈў№╝Ъ

      УАеТЋ░ТЇ«ТЌбтЈ»С╗ЦтГўтюетЁ▒С║ФУАеуЕ║жЌ┤жЄї№╝їС╣ЪтЈ»С╗ЦТў»тЇЋуІгуџёТќЄС╗ХсђѓУ┐ЎСИфУАїСИ║Тў»ућ▒тЈѓТЋ░ innodb_file_per_table ТјДтѕХуџё№╝џ

      1. У┐ЎСИфтЈѓТЋ░У«Йуй«СИ║ OFF УАеуц║уџёТў»№╝їУАеуџёТЋ░ТЇ«ТћЙтюеу│╗у╗ЪтЁ▒С║ФУАеуЕ║жЌ┤№╝їС╣Ът░▒Тў»УиЪТЋ░ТЇ«тГЌтЁИТћЙтюеСИђУхи№╝Џ
      2. У┐ЎСИфтЈѓТЋ░У«Йуй«СИ║ ON УАеуц║уџёТў»№╝їТ»ЈСИф InnoDB УАеТЋ░ТЇ«тГўтѓетюеСИђСИфС╗Ц .ibd СИ║тљју╝ђуџёТќЄС╗ХСИГсђѓ

      С╗ј MySQL 5.6.6 уЅѕТюгт╝ђтДІ№╝їт«Ѓуџёж╗ўУ«цтђ╝т░▒Тў» ON С║єсђѓ

      ТѕЉт╗║У««СйаСИЇУ«║Сй┐уће MySQL уџётЊфСИфуЅѕТюг№╝їжЃйт░єУ┐ЎСИфтђ╝У«Йуй«СИ║ ONсђѓтЏаСИ║№╝їСИђСИфУАетЇЋуІгтГўтѓеСИ║СИђСИфТќЄС╗ХТЏ┤т«╣ТўЊу«Ауљє№╝їУђїСИћтюеСйаСИЇжюђУдЂУ┐ЎСИфУАеуџёТЌХтђЎ№╝їжђџУ┐Є drop table тЉйС╗ц№╝їу│╗у╗Ът░▒С╝џуЏ┤ТјЦтѕажЎцУ┐ЎСИфТќЄС╗ХсђѓУђїтдѓТъюТў»ТћЙтюетЁ▒С║ФУАеуЕ║жЌ┤СИГ№╝їтЇ│Сй┐УАетѕаТјЅС║є№╝їуЕ║жЌ┤С╣ЪТў»СИЇС╝џтЏъТћХуџёсђѓ

      ТЅђС╗Ц№╝їт░є innodb_file_per_table У«Йуй«СИ║ ON№╝їТў»ТјеУЇљтЂџТ│Ћ№╝їТѕЉС╗гТјЦСИІТЮЦуџёУ«еУ«║жЃйТў»тЪ║С║јУ┐ЎСИфУ«Йуй«т▒Ћт╝ђуџёсђѓ

      ТѕЉС╗гтюетѕажЎцТЋ┤СИфУАеуџёТЌХтђЎ№╝їтЈ»С╗ЦСй┐уће drop table тЉйС╗цтЏъТћХУАеуЕ║жЌ┤сђѓСйєТў»№╝їТѕЉС╗гжЂЄтѕ░уџёТЏ┤тцџуџётѕажЎцТЋ░ТЇ«уџётю║ТЎ»Тў»тѕажЎцТЪљС║ЏУАї№╝їУ┐ЎТЌХт░▒жЂЄтѕ░С║єТѕЉС╗гТќЄуФат╝ђтц┤уџёжЌ«жбў№╝џУАеСИГуџёТЋ░ТЇ«УбФтѕажЎцС║є№╝їСйєТў»УАеуЕ║жЌ┤тЇ┤Т▓АТюЅУбФтЏъТћХсђѓ

      ТЈњтЁЦтњїтѕажЎцТЊЇСйютЈ»УЃйС╝џжђаТѕљуЕ║Т┤ъсђѓ

      • ТЈњтЁЦТЌХ№╝їтдѓТъюТЈњтЁЦСйЇуй«ТЅђтюежАхти▓Т╗А№╝їжюђУдЂућ│У»иТќ░жАхжЮбсђѓ
      • тѕажЎцТЌХ№╝їСИЇС╝џтѕажЎцТЅђтюежАх№╝їУђїТў»т░єУ«░тйЋтюежАхжЮбуџёСйЇуй«ТаЄУ«░СИ║тЈ»жЄЇућесђѓ

      ТЅђС╗Ц№╝їтдѓТъюУЃйтцЪТііУ┐ЎС║ЏуЕ║Т┤ътј╗ТјЅ№╝їт░▒УЃйУЙЙтѕ░ТћХу╝ЕУАеуЕ║жЌ┤уџёуЏ«уџёсђѓ

      УдЂУЙЙтѕ░ТћХу╝ЕуЕ║Т┤ъуџёуЏ«уџё№╝їтЈ»С╗ЦСй┐ућежЄЇт╗║УАеуџёТќ╣т╝Јсђѓ
